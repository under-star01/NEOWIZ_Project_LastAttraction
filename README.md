# LastAttraction

**Last Attraction**은 폐쇄된 놀이공원을 배경으로 한  
**1vs4 비대칭 호러 서바이벌 프로젝트**입니다.

생존자는 증거를 수집하고 관제실에 업로드한 뒤 탈출해야 하며,  
추격자는 이들을 추적해 철창에 감금하고 처형하는 것을 목표로 합니다.

현재 저장소의 빌드는 **프로토타입 버전**을 기준으로 정리되어 있습니다.

---

## 프로젝트 개요

- **프로젝트명** : Last Attraction
- **장르** : 1vs4 비대칭 호러 서바이벌
- **개발 인원** : 5인 팀 프로젝트
- **개발 환경** : Unity, C#

---

## 주요 플레이 요소

- 생존자 4인 vs 추격자 1인의 비대칭 구조
- 증거 수집 및 업로드를 통한 탈출 루프
- 추적, 다운, 철창 감금으로 이어지는 추격자 루프
- 판자, CCTV, 구출 알림 등 정보 비대칭 기반 심리전

---

## 프로토타입 구현 상태

- 매칭 시스템
- DB 서버 연동 및 로그인 / 회원가입
- 생존자 / 추격자 기본 플레이 구현
- 공용 오브젝트 상호작용 동기화
- 구속 상태(감옥) 구현
- 프로토타입 맵 배치 및 밸런스 조절

---
